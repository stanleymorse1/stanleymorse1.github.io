Project Eden

T̊̽̒ȟ̑iͩ͡s̥ͣͦ ̸̆͑D͏ig̮i̵̸t̡͞ȁ̮ͩḷ̫̖ ͬP̈́ͭ͠ri͛ş̼̈́o̩n͏͔͠ ̖̺̬wǒn̈́'̡t ͭͦc̩̈ͩǫ̓nͬ̏ͧta͙ͣͩḭ̅ͥn̹ ̻̇u̞s.͆̔҉.̢̩̺.̹̄͒

Project Eden (working title) is a VR survival-horror game, taking place in a mysteriously damaged simulation full of deadly, cunning and adaptive AI abominations.

The game features an experimental VR user interface system, where buttons are physically simulated physics-based objects. The creator tool uses these aspects to create a fluid and fun interface; the scroll bar can be thrown with momentum, objects can be selected by grabbing and pulling them out of the screen, and added or removed from the favourites wheel by twisting clockwise.

The game features exciting VR combat as players earn blueprints from exploring and defeating enemies, which they can use to build weapons, traps, defences and anything else they can think of. Using the creator tool's intuitive controls and interface, weld sharp stakes to a shield, create a makeshift hammer out of a heavy brick and a pole or attach knives to a tracking block to make homing darts to overwhelm enemies in a game where wit matters above all.

Eden is in early development.
